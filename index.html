Academic Projects

  • Spatial StyleGAN      Supervised by Dr.Yujun Shen. 2021
    • Method: We replaced the first feature map of StyleGAN with pixel coordinates encoding(Fourier encoding and SIREN encoding). To avoid the influence from zero padding, we removed all padding, bias and added noises and keep constant margins of all feature maps.
    • Result: We successfully tanslate, scale, crop and outpaint(extend) the generated images by manipulating encoded coordinates.
  • Single-view 3D reconstruction based on SDF.      Supervised by Prof. Ken Perlin. 2019
    • Method: 1. Generate a coarse Signed Distance Function(SDF) from an image, using marching cube to generate a mesh. 2. Deform the coarse network to the fine result.
  • Thesis of Bachelor of Engineering.      Supervised by Prof. Jinlei Jiang. 2018
    • Title: The porting and parallel optimization of a 2-dimensional acoustic wave propagation modeling program based on the poly-grid spectral element method. (Translated from the original Chinese title)
    • Main method: 1. Re-implement the program(Fortran) with C++ and Armadillo (a high performance linear algebra library). 2. Re-implement the bottleneck operators with function from libraries. 3. Accelerate the program with OpenMP and MPI.
    • Result The maximum speedup achieved by the improved version is 29.2 when running on a Linux server with 16 physical cores, compared with the original Fortran code.
